# Práctica 4 — TypeScript

## Objetivo general

El objetivo de esta práctica es aplicar TypeScript en el desarrollo de aplicaciones, utilizando tipado estático para mejorar la calidad, mantenibilidad y escalabilidad del código.

Se busca:

- Detectar errores en tiempo de desarrollo.
- Definir estructuras de datos claras mediante tipos e interfaces.
- Aplicar buenas prácticas como uniones discriminadas y genéricos.
- Organizar el código en módulos independientes.

---

## Estructura

- `modulo-1`: lógica matemática con tipado estricto.
- `modulo-2`: modelado de dominio y arquitectura con genéricos.

Cada módulo incluye su propia documentación en la carpeta `docs/`.

---

## Tecnologías

- TypeScript
- Node.js
